Run with the following command:
cd ~/hailo-apps-infra
source setup_env.sh
cd ~/cat_litter_monitor


python hailo_streamer.py \
  --hef /usr/local/hailo/resources/models/hailo8/yolov8m.hef \
  --input /dev/video0 \
  --use-frame \
  --disable-sync \
  --batch-size 1 \
  --frame-rate 120


View video stream from the following url:

http://192.168.1.231:5000/video_feed/

To make feed viewable over the internet use the following:
sudo tailscale funnel --bg http://127.0.0.1:5000
tailscale funnel status

The feed should now be available at:
https://arthurtracker.tailb5a2e4.ts.net
or the pi tailscale url at port 5000

To disable the public feed:
sudo tailscale funnel --https=443 off
